A modern, fully responsive static website built for a local home bakery. This project emphasizes clean UI/UX design, aesthetic product presentation, and an intuitive user interface, serving as a practical showcase of fundamental front-end web development skills.

---

## 🍰 Key Features

* **Homepage Landing:** An engaging main page featuring an attractive hero banner and a quick summary of the bakery's key offerings.
* **Product Catalog:** Neatly organized categories showcasing birthday cakes, pastries, and daily baked goods.
* **User/Admin Panel (Mockup):** Static subpages demonstrating UI layouts for user profiles, post/content editing, and a dashboard for basic statistics.
* **Contact Form:** A beautifully styled form equipped with JavaScript to enable customer inquiries.
* **Fully Responsive:** Fluid design optimized for seamless viewing across mobile phones, tablets, and desktop monitors.

---

## 🛠️ Technologies Used

* **HTML5:** Semantic document structure ensuring better accessibility and SEO.
* **CSS3:** Modern layout techniques utilizing **Flexbox** and **CSS Grid**, alongside custom fonts, smooth transitions, and keyframe animations.
* **JavaScript (Vanilla):** DOM manipulation, basic interactivity, and form handling without relying on heavy external frameworks.
